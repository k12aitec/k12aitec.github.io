# “中小学人工智能与科技教育大会”官方网站

为落实《国务院关于印发新一代人工智能发展规划的通知》和全国中小学科学教育工作推进会精神，普及中小学人工智能教育，加强科学教育、探索信息科技跨学科融合路径，上海师范大学王龚教授和温州科技高级中学谢作如老师共同发起这个活动。

最初大会的名称为“中小学人工智能与创客教育大会”，是为了正面回答人工智能教育与创客教育二者并不相同。2025年10月23日，教育部等七部门发布《关于加强中小学科技教育的意见》。自2026年开始，大会更名为“中小学人工智能与科技教育大会”。
